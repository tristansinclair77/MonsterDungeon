# Copilot Instructions for Monster Dungeon
# Feature: Fixed aspect ratio + frameless game window

Modify MainWindow.xaml and MainWindow.xaml.cs to create a fixed-size, borderless window optimized for game display.

## Requirements

1. **Window Style**
   - Remove the Windows title bar completely.
   - Set the following properties in `MainWindow.xaml`:
     ```xml
     WindowStyle="None"
     ResizeMode="NoResize"
     AllowsTransparency="False"
     WindowStartupLocation="CenterScreen"
     Background="#FF0D0D0D" <!-- or bound to current theme -->
     ```
   - Optionally set a subtle drop shadow via MahApps.Metro if desired.

2. **Aspect Ratio**
   - Maintain a 16:9 ratio (e.g., 1280x720) regardless of monitor resolution.
   - Lock this aspect ratio dynamically in `MainWindow.xaml.cs` by handling `SizeChanged`:
     ```csharp
     protected override void OnRenderSizeChanged(SizeChangedInfo info)
     {
         const double aspect = 16.0 / 9.0;
         double newWidth = info.NewSize.Width;
         double newHeight = newWidth / aspect;
         if (Math.Abs(info.NewSize.Height - newHeight) > 1)
             Height = newHeight;
         base.OnRenderSizeChanged(info);
     }
     ```
   - This prevents window distortion if users try to resize or maximize.

3. **Center on Screen**
   - Set `WindowStartupLocation="CenterScreen"`.
   - On launch, ensure the size is fixed (1280x720 or 960x540 for smaller windows).

4. **Optional Fullscreen Mode (Debug Toggle)**
   - Add a future-ready command (in DebugMenuViewModel) for “Toggle Fullscreen.”
   - This will switch between:
     ```csharp
     WindowStyle = WindowStyle.None;
     WindowState = WindowState.Maximized;
     ```
     and back to windowed mode (1280x720, centered).

5. **Next Steps Placeholder**
   - Prepare an inner Grid named `GameRoot` that will contain all future UI/HUD layers.
   - It should auto-scale contents if fullscreen is added later:
     ```xml
     <Viewbox Stretch="Uniform">
         <Grid x:Name="GameRoot" Width="1280" Height="720" />
     </Viewbox>
     ```
